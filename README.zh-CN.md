### 原生 iOS + macOS 的 K12 AI 学习助手

**端侧智能优先 · 不绑定任何 AI 厂商 · 一套通用 SwiftUI 代码**

> 用原生 Apple 平台重新打造的 K12 AI 学习应用 —— **集成更紧密、体验更自然、并以端侧智能增强。**
> 开箱即用、**完全离线**（内置一套确定性的本地引擎为每个功能兜底），无需改动任何功能页即可升级到真实的
> 云端或端侧大模型。

## 亮点

- **一套代码，三种形态。** iPhone 用 `TabView`，iPad 与 Mac 用 `NavigationSplitView` —— 同一批功能视图，
  在每个平台上都符合原生习惯；大屏上支持真正的多栏学习。
- **当下即可离线使用。** 默认内置功能完整的 `MockIntelligenceService`：确定性、可演示、无需联网或权限。应用
  **绝不**强依赖任何后端。
- **自带 AI，自由选择。** 可选接入端侧 Apple 基础模型，或你偏好的云端厂商（豆包 / 通义千问 / 智谱 GLM /
  Kimi / DeepSeek / MiniMax / Claude / Gemini / OpenAI）。密钥保存在钥匙串，所有 AI 功能都经由同一套抽象。
- **重在引导，而非直接给答案。** "学习模式"（先尝试、苏格拉底式分层提示、家长验证后才看完整答案）贯穿
  拍照解题、作文批改与豆包老师。
- **本地优先，随时可同步。** SwiftData 为唯一数据源；所有模型兼容 CloudKit，无需迁移即可开启跨设备同步。

## 功能

应用覆盖完整的 K12 学习链路，已实现 **70+ 项功能**。

### 📸 解题与批改
- **拍照搜题** —— 拍摄或导入题目 → 校正 → OCR 识别（印刷体 / 手写 / 公式）→ 可编辑的题目文本。
- **框选单题 / 整页批改** —— 框选单道题，或检测整页所有题目并逐题 ✓/✗。
- **结构化解答** —— 思路 · 步骤 · 答案 · 知识点，配可无障碍朗读的类 LaTeX 公式渲染与图示。
- **实时扫题** —— iOS 上的 VisionKit 实时取词，其它平台优雅回退到拍照解题。
- **作业批改** —— 拍照、相册或文件导入；AI 将**每一道题**批改进与学科无关的 `GradedWorkbook`
  （数学 / 语文 / 英语 / 任意学科），呈现为分数概览 + 逐题 ✓/✗/◐ 卡片（学生答案 vs 正确答案、错因、
  讲解、知识点标签、步骤、评分量规）。在能力足够的模型上走多模态视觉，其它走 OCR 文本，离线时由确定性
  本地引擎兜底。**批改历史**可离线、原样重现任意一次历史批改。

### 🧑‍🏫 辅导与陪伴
- **豆包老师 · 动态黑板辅导** —— 语音优先，黑板与讲解流式生成，按住说话的"是否听懂了"理解闭环，可随时追问，
  按年级分层讲解。
- **AI 伙伴** —— 知识问答（开放域）、成长挚友（安全护栏）、识万物多模态镜头、按意图分发到专项技能、
  可续接的会话历史。
- **声音与语速** —— 方言/口音选择（普通话 / 粤语 / 台湾国语 / English）+ 语速滑杆。

### ✍️ 练习
- **作文批改** —— 综合 + 分句点评、评分、升格建议、朗读、评分雷达图，只做教练不代写。
- **口算批改 / 口算练习** —— 批量口算批改与年级自适应练习。
- **靶向练习** —— 由知识图谱驱动的每日薄弱点训练。
- **听写** —— TTS 朗读、你来书写、OCR 自动批改。
- **背单词** —— 间隔重复（SRS）卡组、卡片与测验。
- **英语口语** —— 情景角色扮演 + 纠错 + 发音评分。
- **课文翻译 · 文言文** —— 段落 OCR → 双语对照 + 注释 + 朗读；古诗文研读、背诵检测、"与诗人对话"。
- **模拟测验** —— 选择学科 / 题量 / 时长 → 实时倒计时 → 自动批改报告，逐题回顾并可加入错题本。

### 📚 知识与课程
- **错题本** —— 收集、归类、复习错题；组卷 / 导出；批量加入题库。
- **错因分析 + 知识图谱** —— 薄弱点映射，可一键进入针对性练习。
- **知识点讲解 2.0** —— 背景 → 内容 → 价值，与个人知识图谱联动。
- **题库** —— 把错题（或任意题目）存入题库以备复习；按学科 / 星标筛选、看答案、让豆包老师讲。内置
  **遗忘曲线复习闭环**（重来 / 困难 / 良好 / 简单 → 掌握度 + 下次复习时间）与**今日复习**到期筛选。
- **智能出题** —— 将题库中的题目生成**同类练习**，并可把生成的题目再次入库，循环复习。
- **豆包课堂 · 定制课程** —— 带时间点测验的互动 AI 课程；按需生成课程。
- **文档/PDF 问答** —— 导入 → 摘要/大纲 + 对话问答。

### 📊 洞察与激励
- **答题足迹** —— GitHub 风格的每日热力图，统计**每一道**作答、批改或练习过的题目（5 级绿色生长色阶、
  按自然周分列、横向滚动并自动定位到今天、月份与星期标签、每格无障碍标注）。中枢 `ActivityRecorder` 将
  所有学习入口汇入同一份记录，使热力图、**连续天数**与报告永不矛盾。出现在 个人中心、今日 与 学习报告。
- **学习报告** —— 时长、掌握度、趋势图表，并给出薄弱点预警。
- **今日** —— 一屏掌握的每日计划：目标环 + 打卡、靶向练习、到期错题与单词、继续学习、7 天时长图，
  全部可深度跳转。
- **成就墙** —— 经验/等级环、连续打卡热区、分级徽章墙（铜/银/金/铂，持久化解锁日期）—— 全部由真实历史实时推导。

### 🧩 框架与平台
- 四标签自适应导航（首页 / 学习 / 工具 / 我的）· 首启引导 + 年级向导 · Sign in with Apple + 游客 ·
  个人中心 · 带搜索的工具中心 · 空/加载/错误/离线状态体系。
- **家长模式** —— 验证关卡、答案隐藏、各项控制、学情报告。
- **统一搜索**、**App Intents / Siri / Spotlight**、**macOS 菜单与快捷键**
  （拍照解题 ⌘N、批改 ⇧⌘G、今日练习 ⌘T、问豆包 ⌘L、搜索 ⌘F、分区 ⌘1–⌘9）、科学计算器 + 公式库、
  以及 专注 · 番茄钟。

## 架构

清晰的分层 SwiftUI 架构，完整蓝图见 [`ARCHITECTURE.md`](ARCHITECTURE.md)。

```
App        DoubaoAiXueApp · RootView · AppShell · AppRouter（类型化路由）
Features   首页 · 解题 · 辅导 · 练习 · 知识 · 课程 · 伙伴
           · 报告 · 作业批改 · 题库 · 测验 · 工具 · 个人中心 · …
Services   IntelligenceService（Mock | 基础模型 | 云端）· OCR/视觉
           · 语音（识别 + 合成）· SRS · ContentCatalog · 计划 · ActivityRecorder
Data       SwiftData @Model 实体 · ModelContainerFactory · SampleData
Design     DBTheme 设计令牌 · DB* 组件 · Liquid Glass · MathText · ViewState
```

- **AI 厂商抽象。** 所有模型调用都经过 `IntelligenceService` 协议，通过 `\.intelligence` 注入。三种实现：
  `MockIntelligenceService`（默认、离线、确定性）、`FoundationModelsService`（端侧 Apple 智能，可用时启用）、
  以及多厂商 `CloudIntelligenceService`。跨边界的 DTO 都是纯 `Sendable` 值类型；功能层再把它们映射为
  SwiftData 模型做持久化。
- **唯一导航真相源。** `AppRouter`（`@Observable`、`@MainActor`）+ 类型化 `Route`；跨功能深链都经由它解析，
  绝不直接引用视图。
- **并发。** 工程级默认 `MainActor` 隔离、Swift 6 严格并发；重活均为 `async` / `Sendable`。
- **构建模型。** 单一 App target + 文件同步分组 —— `豆包爱学/` 下的任意文件都会被自动编译（新增源码无需改
  `.pbxproj`）。

**技术栈：** Swift 6 · SwiftUI · SwiftData · Observation · Swift Charts · Vision/VisionKit ·
Speech/AVFoundation · PencilKit · App Intents ·（可选）FoundationModels。

## 构建与运行

```bash
# 克隆
git clone https://github.com/linroger/doubao-aixue.git
cd doubao-aixue

# 在 Xcode 中打开（scheme：豆包爱学），在模拟器或本机运行
open 豆包爱学.xcodeproj

# …或用命令行构建：
./init.sh            # 构建 macOS（默认）
./init.sh ios        # 构建 iOS 模拟器
```

应用启动即进入一份内容丰富的演示数据（示例档案、错题、卡组、课程、知识图谱，以及约 12 周的活动记录），
并可基于本地引擎**完全离线**使用 —— 无需 API 密钥。

### 启用真实 AI（可选）

打开 **设置 → 多模型 / AI 厂商**，选择厂商并粘贴 API 密钥（保存在钥匙串）。此后所有 AI 功能 —— 解题、辅导、
作业批改、作文、伙伴、文档问答、智能出题 —— 都会路由到你选定的模型；当某次请求无法完成时，会自动回退到
端侧 / 离线。具备视觉能力的模型还会驱动图片版解题与作业批改。

## 发布 macOS 应用（.dmg）

```bash
./scripts/package_dmg.sh          # → dist/豆包爱学.dmg（Release，临时签名，可本机运行）
```

该脚本构建 Release 配置、对应用做临时（ad‑hoc）签名，并打包成可拖拽安装的磁盘镜像。该 DMG
**仅临时签名、未公证**，首次打开请用**右键 → 打开**（macOS 会记住该选择）。公证流程（需付费 Apple 开发者
账号）见 [`RELEASE.md`](RELEASE.md)。

## 隐私

默认情况下，应用在**端侧**处理一切（Mock 或 Apple 基础模型），数据通过 SwiftData 本地存储。云端 AI 严格按需启用：
在你填入厂商密钥之前没有任何数据离开设备，密钥保存在系统钥匙串中。
